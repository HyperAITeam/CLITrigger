# v0.2.16 — "내 일정" 2차 보강 + 세션 멀티 윈도우 복귀

릴리즈 일자: 2026-06-04
이전 버전: v0.2.15

## 요약 (TL;DR)

v0.2.15에서 처음 선보인 전역 **"내 일정"(My Schedule)** 을 실사용에 맞게 크게 보강했습니다. 메모에 이미지를 붙이고, 표 뷰를 색상·정렬로 정리하고, 개인 항목·Jira 이슈를 프로젝트 플래너로 넘기고, Jira 가져오기 기준을 직접 정하고, 쌓인 메모를 기간 단위로 한 번에 비우고, 월 달력의 높이를 조절하거나 빽빽한 날짜를 호버로 펼쳐 볼 수 있습니다. 세션 쪽은 별도 창(멀티 윈도우)으로 빼낸 터미널을 목록에서 한눈에 알아보고 중앙 창으로 다시 불러올 수 있게 했습니다.

## 주요 변경

### 내 일정 — 메모 이미지 & 라이트박스

- 메모 편집기에 **이미지 첨부**(붙여넣기 / 드래그앤드롭 / 파일 선택)와 썸네일 미리보기·삭제를 추가했습니다.
- 내 일정·플래너의 썸네일을 클릭하면 **원본 크기 라이트박스**로 크게 볼 수 있습니다(배경/Esc 닫기).

### 내 일정 — 표 뷰 & 캘린더 조작

- 표 뷰의 **종류·상태를 색상 배지**로 표시하고, 컬럼 헤더(날짜/종류/상태)를 클릭해 정렬할 수 있습니다.
- 달력 셀 안의 항목 칩을 **클릭하면 바로 열립니다**(메모=편집, 스케줄/플래너=프로젝트 이동, Jira=새 탭).
- 월 달력은 아래 핸들을 **드래그해 높이(크기)** 를 조절하고, 항목이 많아 `+N`으로 접힌 날짜에 잠깐 **마우스를 올리면 그 날짜가 카드로 확대**됩니다.
- 플래너에도 월/주/일/표 캘린더 뷰가 추가됐습니다.

### 내 일정 — 플래너 이동 / Jira 기준 / 기간 삭제

- 개인 메모나 Jira 이슈를 **프로젝트 플래너로 이동/추가**할 수 있습니다(메모는 이미지·태그·마감일까지 옮기고 원본 삭제, Jira 이슈는 그대로 둔 채 복제).
- Jira **가져오기 기준을 직접 설정**: '나에게 할당만' / '완료 포함' 토글, 프로젝트 키 제한, 고급 JQL.
- 헤더의 **휴지통** 버튼으로 특정 기간의 개인 메모를 **한 번에 삭제**합니다(완료된 항목만 / 날짜 없는 백로그 포함 옵션, 삭제 대상 개수 미리보기).

### 세션 — 멀티 윈도우(별도 창) 상태 표시 + 중앙으로 불러오기

- 터미널을 별도 OS 창으로 빼내면 목록 행에 **별도 창 배지**가 표시되고, **중앙으로 불러오기** 버튼이나 행 클릭으로 메인 창 안 플로팅으로 복귀합니다(세션·출력 유지).
- 별도 창은 하단 독에도 칩으로 나타나며, 칩을 클릭하면 중앙으로 불러옵니다.
