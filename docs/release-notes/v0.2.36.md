# v0.2.36 — v0.2.35 재배포 (릴리스 파이프라인 복구)

Release date: 2026-07-20
Previous version: v0.2.34

## TL;DR

v0.2.35의 GitHub 릴리스가 배포 후 유실되어 자동 업데이트가 동작하지 않는 문제를 해결하기 위한 **재배포 릴리스**입니다. 코드 변경은 없으며, 내용은 [v0.2.35](v0.2.35.md)와 동일합니다: 앱 내 Git/SVN 충돌 해결 UI, 터미널 개선(집중 레이아웃, 팝아웃 설정 유지, 한글 IME 자가 복구, ASCII 아트 렌더링), 오류 토스트 정리.

자세한 변경 사항과 전체 커밋 목록은 [v0.2.35 릴리스 노트](v0.2.35.md)를 참고하세요.

## Install / Update

```bash
npm i -g clitrigger@0.2.36
```

Desktop app — pick the right asset:

- **Windows (recommended)**: `CLITrigger-Setup-0.2.36.exe` — installer with Start Menu shortcut, uninstaller, and auto-update.
- **Windows (portable)**: `CLITrigger-0.2.36.exe` — run directly, no install; no shortcuts and no auto-update (grab new versions manually).
- **macOS**: `.dmg` / **Linux**: `.AppImage`
