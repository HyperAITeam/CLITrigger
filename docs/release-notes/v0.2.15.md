# v0.2.15 — 내 일정(My Schedule) + Jira 연동, 자동화 허브 탭, 설정 재구성

릴리즈 일자: 2026-06-03
이전 버전: v0.2.14

## 요약 (TL;DR)

- **내 일정(My Schedule)**: 프로젝트와 무관한 전역 개인 캘린더/메모가 사이드바에 새로 생겼습니다. 월/주/일/표 뷰, 태그, 노션식 메모 에디터에 더해 **전 프로젝트 스케줄·플래너 마감일과 내게 할당된 Jira 작업**을 한 화면에 모아 봅니다.
- **즐겨찾기**: 실행파일을 경로 타이핑 대신 **파일 탐색기로 골라** 등록할 수 있습니다(웹/앱 공통).
- **자동화 허브**: 자동 작업·토론·스케줄을 하나의 **"자동화" 탭**으로 묶고, 프로젝트 설정을 **실행/워크스페이스**로 재구성했습니다.
- 사이드바 접기, 팝아웃 터미널 한글 입력(Windows) 복구, SVN diff 오류 수정 등 다듬기.

## 주요 변경

### 내 일정 (My Schedule) — 신규

- 사이드바 **내 일정**(`/agenda`)에서 여는 전역 개인 캘린더. 실행은 하지 않는 순수 정리 공간입니다.
- **월 / 주 / 일 / 표** 4가지 뷰. 날짜 칸 hover 시 **"+"** 로 그 날짜에 바로 메모 추가.
- **메모 에디터**: 제목 선택사항(비우면 본문 첫 줄), 좌상단 **크게 보기**로 전체화면 페이지 뷰, 색상 **태그** + 태그 필터.
- 개인 항목과 함께 **전 프로젝트의 스케줄·플래너 마감일**을 읽기 전용으로 겹쳐 표시(클릭 시 해당 프로젝트로 이동). 캘린더↔우측 패널 너비는 드래그로 조절됩니다.

### Jira 연동

- 내 일정 전용 **Jira 연결**(톱니바퀴 → 사이트 주소·이메일·API 토큰)을 설정하면, 내게 할당된 작업이 캘린더에 겹쳐 보입니다.
- 기한 있는 이슈는 날짜에, 기한 없는 이슈는 **"Jira (마감일 없음)"** 목록에 표시. 클릭하면 Jira로 이동하고, **"내 일정에 추가"** 로 편집 가능한 메모로 가져올 수 있습니다.
- 비개발자도 따라할 수 있게 **API 토큰 발급 링크 + 단계 안내 + 연결 테스트**를 모달에 넣었습니다.
- Jira Cloud가 제거한 옛 검색 API 대응으로 새 엔드포인트(`/search/jql`)로 전환 — **프로젝트별 Jira 패널**의 이슈 목록도 함께 정상화됐습니다.

### 즐겨찾기 — 파일 탐색으로 등록

- 실행파일(exe/.bat/.sh) 등록 시 대상 칸 옆 **"찾아보기"** 버튼으로 OS 파일 탐색기에서 바로 선택. 워크스페이스 폴더를 고르는 방식과 동일하게 웹 브라우저에서도 동작합니다.

### 자동화 허브 탭 + 프로젝트 설정 재구성

- **자동화 탭**: 흩어져 있던 자동 작업 / 토론 / 스케줄을 하나의 "자동화" 탭 + 내부 서브탭으로 통합(기존 딥링크 호환).
- **프로젝트 설정**을 **실행 설정**(CLI/모델/워커·턴/샌드박스/워크트리 등)과 **워크스페이스**(메모리 자동수집·SVN)로 분리해 정리.
- 자동 작업 전용이던 **전체 실행/전체 중지** 버튼을 자동 작업 탭 안으로 이동.

### 기타 개선 & 수정

- 데스크톱에서 **사이드바를 아이콘 레일로 접기**.
- **팝아웃 터미널 한글 입력(Windows)** 복구 — 프레임리스 창에서 IME가 안 먹던 문제 수정.
- **SVN** diff/커밋의 "not a working copy" 오류 수정.
- 터미널에서 **선택 영역 Ctrl/Cmd+C 복사**(선택 없으면 기존처럼 ^C 전송), Git 툴바 상태 배지.
