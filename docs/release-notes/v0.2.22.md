# v0.2.22 — 터널 안정화 · SVN/문서/플래너 UX 다듬기

릴리즈 일자: 2026-07-01
이전 버전: v0.2.21

## 요약 (TL;DR)

- 패키징된 데스크탑(exe)에서 cloudflared 터널이 실행되지 않던 문제를 수정했고, 실패 시 원인을 에러 메시지에 그대로 노출하도록 했습니다.
- SVN 변경사항을 디렉터리 트리로 묶어 보여주고(flat 토글 제공), 프로젝트별 로컬 상태를 캐시해 재방문 시 즉시 뜨도록 했습니다.
- 자동화 허브 서브탭에 호버 도움말이 붙었고, 문서·플래너·터미널의 크고 작은 UX가 다듬어졌습니다.

## 주요 변경

### 터널 (Cloudflare)

- 패키징(exe) 환경에서 cloudflared가 실행되지 않던 문제를 수정했습니다. 이제 데스크탑 앱에서도 터널이 정상적으로 뜹니다.
- 터널 기동에 실패하면 cloudflared의 마지막 출력을 에러에 첨부해, 무엇 때문에 실패했는지 바로 확인할 수 있습니다.

### SVN

- 변경사항 목록을 평면 나열 대신 디렉터리 트리로 그룹화해 한눈에 파악하기 쉬워졌습니다. 기존 평면 보기가 필요하면 flat 토글로 되돌릴 수 있습니다.
- 로컬 상태를 프로젝트별로 캐시합니다. 탭을 다시 열 때 매번 새로 조회하지 않고 즉시 표시됩니다.

### 자동화 허브

- 작업/토론/루틴/통계 서브탭에 호버 도움말을 추가해, 각 탭이 무슨 역할인지 마우스를 올리면 안내됩니다.

### 문서(Vault)

- 볼트 탭 명칭을 "문서"로 통일하고 옵시디언 관련 언급을 정리했습니다.
- 리사이저를 포인터 이벤트로 전환해, iframe 위에서 드래그하다 놓치는 현상을 방지했습니다.

### 플래너 · 터미널

- 플래너 사이드바에 드래그 리사이즈 핸들을 추가해 폭을 자유롭게 조절할 수 있습니다.
- 터미널의 조합/이미지 미리보기 오버레이를 오른쪽 아래로 옮기고 IME 라벨을 제거했습니다.
