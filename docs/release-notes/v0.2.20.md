# v0.2.20 — 플래너에 Notion 스타일 페이지 워크스페이스

릴리즈 일자: 2026-07-01
이전 버전: v0.2.19

## 요약 (TL;DR)

플래너가 단순 할일 목록을 넘어 **Notion 스타일 페이지 워크스페이스**로 확장됐습니다. 좌측 사이드바에서 자유형식 페이지를 만들고, 본문에 `/`로 **할일 목록·캘린더 블록**을 끼워넣어 그 페이지 전용 작업을 관리할 수 있습니다. 회의 노트·프로젝트·위클리 **템플릿**으로 빠르게 시작할 수 있고, 통계 탭은 자동화 허브로 통합됐습니다. 이 외에 Windows 한글 IME 입력 멎음, Vault/SVN/도크 관련 다수 개선이 포함됩니다.

## 주요 변경

### 플래너 — Notion 스타일 페이지

- 플래너에 **페이지** 개념이 추가됐습니다. 좌측 통합 사이드바에서 페이지를 만들고 BlockNote 기반 에디터로 자유롭게 작성합니다(굵게/체크리스트/제목 등, `/` 슬래시 메뉴).
- 페이지 본문에 **할일 목록 블록**과 **캘린더 블록**을 임베드할 수 있습니다(`/할일목록`, `/캘린더`). 여기서 만든 작업은 **그 페이지 전용**이며, 마감일이 있으면 "전체 보기" 달력/목록에도 롤업됩니다.
- 임베드된 작업도 기존처럼 **터미널(세션)·자동화(작업/루틴)로 변환**할 수 있습니다.
- 할일·캘린더 블록은 **드래그로 크기 조절**이 가능하고, 조절한 크기는 저장됩니다.
- 새 페이지 생성 시 **템플릿 선택**(빈 페이지 · 회의 노트 · 프로젝트 플랜 · 위클리 플래너). 위클리 플래너는 요일별 체크리스트로 구성됩니다.
- 기존 달력/목록 뷰는 "전체 보기"로 사이드바에 통합됐습니다.

### 자동화 허브 정리

- **통계(Analytics)** 탭이 자동화 허브의 서브탭(작업/토론/루틴/통계)으로 이동했습니다. 기존 `?tab=analytics` 딥링크는 그대로 동작합니다.
- 자동화 "스케줄" 표기를 **"루틴"**으로 변경했습니다.
- 토론을 **워크트리에서 실행할지 여부**를 토론별로 선택할 수 있습니다. 워크트리 없이 실행한 토론의 cleanup이 프로젝트 원본을 건드리지 않도록 수정했습니다.

### 프로젝트 설정 · UI

- 프로젝트 설정에서 **하네스 탭**을 실행설정 탭보다 앞에 배치했습니다.
- 깃 **커밋 메시지 입력란**을 테마 변수 기반 카드 스타일로 바꿔 입력 영역이 또렷하게 보이도록 했습니다.
- 로그인 화면에 **로그인 상태 유지** 체크박스를 추가했습니다.
- 모든 버튼에 외곽 테두리를 적용해 통일감을 높였습니다.

### Vault · SVN

- Vault 그래프에 **PDF 파일도 노드로 표시**되고, HTML 미리보기에서 Ctrl/Cmd+휠 줌을 지원합니다.
- SVN에 **워킹카피/파일 속성 보기 다이얼로그**를 추가하고, 리비전 diff의 status·repo-URL peg 처리를 보강했습니다.

### 버그 수정

- Windows에서 **한글 IME 입력 멎음·조합 창 고착** 문제를 수정했습니다.
- 화면 배율 120% 등에서 도크/팝아웃 좌표가 어긋나던 문제(`screenToClient` 보정)를 수정했습니다.
- 팝아웃→메인 도크 거부 버그, 프로젝트 전환 시 이전 프로젝트 깃 화면이 남던 문제를 수정했습니다.
- Codex 실행에 `--skip-git-repo-check`를 추가해 신뢰되지 않은 워크트리/경로에서 토론이 실패하던 문제를 해결했습니다.
